CSS Pseudo-selectors Information

Structural Pseudo-classes:

  • :first-child: Selects the first child of its parent.
  • :last-child: Selects the last child of its parent.
  • :nth-child(n): Selects elements based on their position among their siblings.
  • :only-child: Selects elements that are the only child of their parent.
  • :first-of-type: Selects the first occurrence of an element type among its siblings.
  • :last-of-type: Selects the last occurrence of an element type among its siblings.
  • :nth-of-type(n): Selects elements based on their position among siblings of the same type.

Link Pseudo-classes:

  • :link: Selects unvisited links.
  • :visited: Selects visited links.
  • :hover: Selects elements when the mouse is over them.
  • :active: Selects elements being activated (e.g., clicked).
  • :focus: Selects elements that have focus.

Negation Pseudo-class:

  • :not(selector): Selects elements that do not match the specified selector.

Pseudo-elements:

  • ::before: Inserts content before the actual content of an element.
  • ::after: Inserts content after the actual content of an element.

These pseudo-selectors provide a way to apply styles to elements in a more dynamic and targeted manner, enhancing the design and interactivity of a web page.
