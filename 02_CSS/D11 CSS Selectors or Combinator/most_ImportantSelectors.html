Most Important used CSS selectors

* Universal Selector

It will target every html selector and we use * icon as *{}

. Class Selectors

The class selector (.) is used to select and style HTML elements that have a specific class attribute.

# ID Selector

The ID selector (#) is used to select and style a specific HTML element with a unique identifier.

> Child Selector (immediate children)

The child selector (>) is used to select and style the direct children of a specified parent element
